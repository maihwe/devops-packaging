# DevOps Software Packaging Project

Demonstrates software packaging practices across three ecosystems: Node.js, Django (Python), and Spring Boot (Java). Each sub-project covers dependency management, semantic versioning, environment separation, distributable artifacts, verification, and security auditing.

- [`nodejs-app/`](./nodejs-app) — Express.js REST API, packaged with npm
- [`django-app/`](./django-app) — Django REST API, packaged with pip
- [`springboot-app/`](./springboot-app) — Spring Boot REST API, packaged with Maven

See each folder's own `README.md` for full details on packaging commands, versioning strategy, and dependency audit results.
